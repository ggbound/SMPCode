## ⌨️ 快捷键

### 全局快捷键
| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl+Shift+P` | 命令面板 | 打开/关闭命令面板 |

| 快捷键 | 功能 | 说明 |

## 🔍 全局搜索

### 打开搜索面板
1. 点击 ActivityBar 中的 🔍 图标
2. 或使用快捷键 `Ctrl+Shift+F`

### 搜索功能
- **搜索框**: 输入要搜索的内容
- **替换框**: 点击 "Replace" 按钮显示替换框
- **文件过滤**: 
  - Include: 指定要搜索的文件类型（如 `*.ts`）
  - Exclude: 排除某些文件（如 `node_modules`）

### 搜索结果
- 按文件分组显示
- 匹配内容高亮显示
- 点击结果跳转到对应行

## 🎛️ 命令面板

### 打开命令面板
- 快捷键: `Ctrl+Shift+P` 或 `Cmd+Shift+P` (macOS)

### 使用方式
1. 输入命令名称进行模糊搜索
2. 使用 ↑↓ 键导航
3. 按 Enter 执行选中的命令
4. 按 Esc 关闭面板

### 可用命令
- **File**: New File, Open File, Save
- **View**: Toggle Terminal, Toggle Search, Toggle Explorer
- **Preferences**: Open Settings
- **Editor**: Split Editor Right, Split Editor Down

## 📊 StatusBar（状态栏）

位于应用底部的信息栏，显示以下内容：

### 左侧区域
- 🔀 **Git 分支**: 当前 Git 分支名称
- ❌ **错误数**: 代码错误数量
- ⚠️ **警告数**: 代码警告数量
- **权限模式**: Read-Only / Workspace Write / Full Access

### 中间区域
- 💰 **成本估算**: Token 使用费用
- **In**: 输入 Token 数量
- **Out**: 输出 Token 数量

### 右侧区域
- **光标位置**: Ln X, Col Y（行号，列号）
- **语言模式**: 当前文件的编程语言
- **编码**: UTF-8
- **行尾**: LF / CRLF
- **缩进**: Spaces: 2 / Tabs
- ⚙️ **设置**: 点击打开设置
- 🔔 **通知**: 通知中心

## 📝 编辑器功能

### Monaco Editor 特性
- ✅ 语法高亮
- ✅ 智能补全
- ✅ 代码折叠
- ✅ 迷你地图（Minimap）
- ✅ 括号配对
- ✅ 自动缩进
- ✅ 多光标编辑

### 标签页管理
- **单击文件**: 预览模式（斜体标签）
- **双击文件**: 固定标签
- **中键点击标签**: 关闭标签
- **拖拽标签**: 调整顺序
- **右键菜单**: 关闭其他/关闭全部/关闭右侧/关闭左侧

### 文件操作
- **保存**: `Ctrl+S`
- **重命名**: 在文件树中右键 → Rename
- **删除**: 在文件树中右键 → Delete
- **新建文件**: 在文件树中右键 → New File
- **新建文件夹**: 在文件树中右键 → New Folder

## 💬 AI 聊天

### 发送消息
1. 在右侧聊天区域输入问题
2. 点击 Send 或按 Enter 发送
3. 等待 AI 回复

### 聊天模式
- **Chat Mode**: 对话式交流
- **Agent Mode**: 自主执行任务

### 权限模式
- **Read-Only**: 只读模式，AI 不能修改文件
- **Workspace Write**: 工作区写入，AI 可以修改项目文件
- **Full Access**: 完全访问，AI 可以执行任意操作

## 🔧 常见问题

### Q: 如何切换侧边栏？
A: 点击 ActivityBar 中的对应图标，或使用快捷键：
- 文件树: `Ctrl+Shift+E`
- 搜索: `Ctrl+Shift+F`

### Q: 如何保存文件？
A: 
- 快捷键: `Ctrl+S`
- 或在命令面板中输入 "Save"

### Q: 如何分屏编辑器？
A: 
- 左右分屏: `Ctrl+\`
- 上下分屏: `Ctrl+K Ctrl+\`
- 或在命令面板中搜索 "Split"

### Q: 如何查看 Git 状态？
A: 查看 StatusBar 左侧的分支信息（🔀 branch-name）

### Q: 如何打开设置？
A: 
- 快捷键: `Ctrl+,`
- 或点击 StatusBar 右侧的 ⚙️ 图标
- 或在命令面板中输入 "Settings"
